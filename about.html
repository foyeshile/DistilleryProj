INST377 - Group 7: Product Discovery Platform

Whisky Distilleries and Ratings

Main About Documentation

Project Information

Our client has tasked us with creating a dynamic and user-friendly product discovery platform. This platform utilizes manufacturer, rating, and price data to provide users with answers that facilitate product exploration, comparison, and decision-making.

Target Audience

Consumers are the primary stakeholders, seeking valuable insights before making purchasing decisions.

How It Works

Users can select a country and see the average rating of each distillery in the country in comparison to the number of whisky varieties. They can also view the Winning bids from a selection of random auction sites.

API Choice

We chose Whisky Hunter's API as it provides a large amount of data of all sorts of kinds. It has auction data about whiskies like prices and sales numbers, as well as having information on distilleries like country of origin, how many whiskies they produce, and their score from votes.

Front-end Technology

The application will be developed using HTML, CSS, and JavaScript. HTML and CSS will be used for formatting, while JavaScript will handle API integration and calculations.
