Whiskey Distilleries and Ratings

Main About Documentation

API Information

Include details about the API's used.

The API from the auction side "Whisky Hunter" was utilized, allowing for information on auctions and distilleries in multiple countries to be analyzed.

Libraries Used

List the libraries and frameworks used in the project w/ brief descriptions.

A library that is being used for this project is Chart.JS. When a user selects a country to see distillery ratings, the scatter chart shows the relationship between that country's distillery's rating by the number of whiskey varieties they produce

NPM Commands

Provide NPM commands used for managing dependencies, building, or running the project.

npm install chart.js was used to get the Chart.JS library.
